# YouTube to MP3 GUI Tool
GUI tool to download a single or list of YouTube videos as MP3s
Utilizes PySimpleGUI and youtube_dl

![App Screenshot](img/screenshot.png)

## Install and Run
- `$ git clone git@github.com:techb/YT-MP3-DL.git`
- `$ pipenv shell`
- `$ pipenv install`
- `$ python yt-to-mp3.py`

## Requires
- PySimpleGUI
- youtube_dl
